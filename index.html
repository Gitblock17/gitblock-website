Questions & Answers

Everything about GitBlock

Is this actually free?

Yes — completely free. No API keys, no credit cards, no subscriptions. GitBlock is MIT licensed open source. You can even run your own inference node.

What models are available?

The network supports multiple frontier models. Access depends on available nodes. We're constantly expanding coverage. Check the Console for the current model list.

How is this different from OpenAI or Anthropic?

No corporation controls GitBlock. No API keys. No usage limits. No training on your data. It's a decentralized network — inference runs on community nodes, not a central server farm.

What is $GITBLOCK?

$GITBLOCK is the native token on Base chain. It powers network governance, staking, and node rewards. Track it on the Gitblock Scanner at gitblock.org/gitblock.

Can I run my own node?

Absolutely. The entire stack is MIT licensed. Clone the repo, run a node, join the network. Your compute contributes to the decentralized inference pool.
