Questions & Answers

Everything about the GitBlock scanner

Is this actually free?

Yes — completely free. No wallet connect, no subscriptions. The scanner runs on open APIs (DEX Screener, GeckoTerminal, GoPlus) and is open source. You can even run your own instance.

How fast are new tokens detected?

The server polls every 15 seconds. New tokens typically appear within 1-2 minutes of their first trade on Base. The trending section updates in real-time.

How does scam detection work?

We use GoPlus Security API which checks every token for: honeypot code, buy/sell taxes, ownership renunciation, mint capabilities, proxy contracts, and blacklists. Results are color-coded from low to critical risk.

What launchpads are tracked?

Clanker, Bankr, Zora, and Virtuals Protocol. We detect the launchpad using on-chain creator addresses and DEX patterns. Coverage expands as new launchpads emerge on Base.

Can I run my own instance?

Absolutely. The entire project is MIT licensed on GitHub. Clone the repo, run npm install && node server.js, and open scanner.html. Deploy to Render or your own VPS in minutes.
