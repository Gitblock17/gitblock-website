How the Network Operates

Four steps from request to response — fully decentralized

1

You Send a Prompt

Through the API, CLI, SDK, or web interface. Your request is encrypted and signed with your wallet.

2

Network Routes It

The gateway finds the best available node based on speed, reputation, model availability, and current load.

3

Node Processes It

A community operator's GPU runs inference on the requested model. The response is encrypted and returned.

4

Verified & Recorded

The interaction is hashed on-chain. Node reputation updates. Quality is scored. The operator earns rewards.

How to Use GitBlock

Code examples for every common use case — copy, paste, run

Chat Completion

Have a conversation with any model. Supports system prompts, multi-turn chat, and all standard parameters.

from gitblock import GitBlock client = GitBlock(api_key="gb_free_xxx") response = client.chat( model="llama-3.3-70b", messages=[ {"role": "system", "content": "You are a helpful coding assistant."}, {"role": "user", "content": "Explain recursion with examples."} ], temperature=0.7, max_tokens=1000 ) print(response.content)

Streaming Responses

Get tokens as they're generated. Build real-time chat UIs, live code assistants, and interactive tools.

from gitblock import GitBlock client = GitBlock(api_key="gb_free_xxx") stream = client.chat( model="mistral-7b", messages=[ {"role": "user", "content": "Write a haiku about code."} ], stream=True ) for chunk in stream: print(chunk.delta, end="", flush=True)

Code Generation

Generate, review, and refactor code using specialized coding models. Supports 50+ programming languages.

from gitblock import GitBlock client = GitBlock(api_key="gb_free_xxx") response = client.chat( model="deepseek-coder-33b", messages=[{ "role": "user", "content": "Write a Python function that " "validates email addresses" }] ) print(response.content)

REST API (cURL)

Call from any language using HTTP. Works with cURL, fetch, axios, requests, or any HTTP library.

curl -X POST https://api.gitblock.io/v1/chat/completions \ -H "Authorization: Bearer YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{ "model": "llama-3.3-70b", "messages": [ {"role": "user", "content": "Hello!"} ] }'

JavaScript / Node.js

Full TypeScript support. Works in Node.js, Deno, Bun, and browser environments.

import { GitBlock } from "gitblock-sdk"; const client = new GitBlock({ apiKey: "gb_free_xxx", }); const response = await client.chat({ model: "llama-3.3-70b", messages: [ { role: "user", content: "Summarize this article" } ], }); console.log(response.content);

Run a Node

Earn rewards by contributing your GPU power. Any hardware works — from laptops to server racks.

# Install node software pip install gitblock-node # Register your node (one-time) gitblock-node register --wallet 0xYourAddr # Start serving inference gitblock-node start --model llama-3.3-70b # Check your stats gitblock-node status # Withdraw earned rewards gitblock-node withdraw

Questions & Answers

Everything you need to know before getting started

Is this actually free? What's the catch?

Truly free. No credit card, no trial period, no bait-and-switch. Node operators contribute GPU power and earn blockchain-based rewards. Users pay nothing. The network is sustained by the community, not by charging you.

Why do I need to connect a wallet?

Wallet connection proves you're a unique human without requiring email, phone, or personal info. It prevents abuse while keeping you anonymous. No funds are moved. No transaction is needed. You just sign a free message.

What models can I use?

89+ open-source models including Llama 3.3, Mistral, DeepSeek Coder, Qwen 2.5, Gemma, Phi-3, CodeLlama, and more. All free. New models are added weekly based on community votes.

Is my data private?

Yes. Every prompt is encrypted before it leaves your device. Only the serving node can decrypt it. We never store, log, or analyze your queries. Zero-knowledge proofs verify quality without exposing content.

Can I replace OpenAI with GitBlock?

Yes. The API uses the same request and response format. Change your base URL to api.gitblock.io, swap in your free API key, and everything works. Same SDK methods, same parameters, same output structure.

How do I earn by running a node?

Install the node software, register your hardware, and start serving models. You earn rewards proportional to the requests you handle and your reputation score. Better hardware and uptime means more traffic and more rewards.

Can I run my own private network?

Absolutely. Everything is MIT licensed. Fork the repository, configure your gateway and nodes, deploy on your own infrastructure. Full documentation covers self-hosted setup for teams and organizations.
